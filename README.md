MotorPH Payroll System


Project Description

The MotorPH Payroll System is a Java-based program that processes employee information and attendance records to compute payroll. 
The system reads data from CSV files and calculates employee salary based on attendance and employee details.

Files Included

MotorPHemployees.csv –  Employee information such as employee ID, name, and salary details.

MotorPHemployeeattendance.csv – Employee attendance records used to calculate working hours.

MotorPHPayroll.iml – IntelliJ project file.
